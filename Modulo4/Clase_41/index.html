🍪 Clase 41: Cookies y Sesiones

Ejemplo 1: Login con Cookies Manuales

¿Cómo funciona?
El nombre del usuario se guarda directamente en la cookie del navegador.

res.cookie("usuario", "Fernando") → El navegador almacena: usuario=Fernando
Resultado aparecerá aquí...

Ejemplo 2: Login con Express-Session

¿Cómo funciona?
Los datos se guardan en el servidor. El navegador solo recibe un ID de sesión.

req.session.usuario = "Fernando" → Servidor guarda los datos
El navegador solo recibe: connect.sid=abc123...
Resultado aparecerá aquí...

📝 Instrucciones

  1. Asegúrate de que el servidor esté corriendo: node src/app.js
  2. Ejemplo 1 (Cookies Manuales): Puedes probar desde Live Server (http://127.0.0.1:5500)
  3. Ejemplo 2 (Express-Session): Debes acceder desde http://localhost:3000/index.html (mismo origen)
  4. Prueba el flujo: Login → Dashboard → Logout → Dashboard
  5. Observa las cookies en DevTools: Application → Cookies
🔍 ¿Qué observar en DevTools?

Ejemplo 1 (Cookie Manual): Verás una cookie llamada usuario con el valor real (ej: "Fernando")

Ejemplo 2 (Session): Verás una cookie llamada connect.sid con un ID largo encriptado
